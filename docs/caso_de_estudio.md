# Caso de estudio MELFA

Este proyecto toma un notebook académico de cinemática de un robot MELFA de 6 GDL y lo convierte gradualmente en un proyecto organizado y reusable.

## Objetivos

- aprender Git y GitHub
- practicar modularización
- construir código reusable
- introducir pruebas básicas
- preparar una base extensible para cinemática y robótica

## Estado inicial

Actualmente el proyecto contiene:
- estructura mínima del repositorio
- README inicial
- carpeta de notebooks
- carpeta src para módulos reutilizables
