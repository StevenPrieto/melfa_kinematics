# MELFA Kinematics

Repositorio de estudio para organizar y modularizar un notebook de cinemática de un robot industrial MELFA de 6 GDL.

## Objetivo

Aprender:
- Git
- GitHub
- modularización
- pruebas básicas
- ambientes virtuales
- organización de proyectos técnicos en Python

## Estructura inicial

- docs/: documentación del caso de estudio
- notebooks/: notebooks originales y exploratorios
- src/: código reusable del proyecto
